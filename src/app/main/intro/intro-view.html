Introduction

Pick your Premier League football dream team amongst an array of players and submit to the database to compare with teams submitted by other users.

App Development

This project demonstrates the following front-end technologies and techniques:

  • AngularJS fundamentals and best practices
  • Firebase for remote database purposes
  • Implemented grid via Susy and Sass
  • Clean, modular, and scalable CSS via the SMACSS pattern

Please note that although the app has a responsive grid, adapting the presentation for small screens has not been prioritised.

Supported Browsers: IE10+ and modern browsers.
